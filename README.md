# oh-my-posh
Customized Oh My Posh theme. This is basically a customized version of the default [powerline](https://ohmyposh.dev/docs/themes#powerline) + [paradox](https://ohmyposh.dev/docs/themes#paradox) themes.

See my blog post for background: https://bradleycarey.com/posts/customizing-my-cli-using-oh-my-posh/

Github gist that I originally used was here: https://gist.githubusercontent.com/avianbc/fc7f8a79633a0e1ca3961e344bbc5f2d/raw/e94ae36cecefa43b70da4a0d56271dfe869e0f2e/avian.omp.json
